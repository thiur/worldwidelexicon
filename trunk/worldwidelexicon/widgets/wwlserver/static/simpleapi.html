Simple Translation API

In celebration of Translator Day, we are publishing a super simple web API that you can use to add machine and human translations to your site. It is as simple as a server side include. Here is the time honored "Hello World" example.

NOTE: if you encounter a Unicode error, please email bsmcconnell at gmail with the error message.

Hello World

To request a human or machine translation to insert in your site, simply call the web API:

www.worldwidelexicon.org/sourcelanguage/t/targetlanguage/encodedtext

or, for example:

www.worldwidelexicon.org/t/en/es/Hello+World

The web API will reply with HTML and Javascript to insert in your page. You will see the translation, plus a link to a popup editor.

Fancy Pants

This example is simple, but you can do much more. You can call the web service at www.worldwidelexicon.org/t/xx/yy with the following CGI parameters:

  • NOTE: xx = source language code, yy = target language code
  • st (source text, UTF-8 encoding)
  • edit=y/n (allow users to edit)
  • allow_machine=y/n (display or hide computer translations)
  • allow_anonymous=y/n (display or hide anonymous translations)
  • output=html, text (if output=text, returns plain text without HTML)

Complete Web API

And, if you want to build a more sophisticated translation application where users can edit translations, submit scores and write comments about translations, see our full web API at www.worldwidelexicon.api
