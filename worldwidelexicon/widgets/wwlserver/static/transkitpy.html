TransKit/Py

TransKit/Py implements the Worldwide Lexicon translation memory in the popular Python programming language. This library uses the standard library, along with the memcache.py module (included in the ZIP file). It supports the following WWL services:

  • Fetch the best available translation for a text, supports machine translation, user translations and professional translations.
  • Submit a translation to the translation memory.
  • Score a translation.
  • Create a new WWL user.
  • Authenticate a user.
  • Get/set user attributes (e.g. city=Denver)
When requesting translations, the get() function looks at sources in the following order:

  • Check memcached (if it is running) to see if a translation is cached.
  • Check gettext (if it is installed) to see if the text is translated in a localization file.
  • Call WWL to request the best available human or machine translation.

Example Usage

translation = wwl.get('en','es','Hello World')

result = wwl.submit('en','es','Hello World','Hola Mundo')

professional_translation = wwl.get('en','es','Hello World',lsp='speaklike', lspusername='foo',lsppw='bar')
