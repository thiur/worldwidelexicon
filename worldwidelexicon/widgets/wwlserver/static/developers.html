For Developers

The Worldwide Lexicon is an open source, collaborative translation memory that combines translations from a variety of sources including machine translation, volunteers, and optionally professional translators. The system works in a manner similar to Wikipedia and allows people to create, edit and score translations via an intuitive user interface.

The WWL translation memory is written in Python, a powerful and easy to use object oriented programming language, and runs on App Engine, Google's highly scalable and cost effective distributed computing environment.

Web API

WWL provides a simple and well documented web API that developers can use to embed translation services into virtually any web app or publishing environment. With just a few web API calls, you can embed collaborative translation in virtually any web app or system. Full documentation and test HTML forms are available here.
