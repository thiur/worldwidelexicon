WWL For Drupal

WWL For Drupal is an interactive translator that adds human/machine translation to Drupal websites. This tool works in a similar manner to the Word Press Translator and Translation Proxy Server. It inserts translations into pages as they are served, and uses the best available machine translations, and translations from volunteers or professional translators. The add on provides the following features:

  • Machine translations from popular web translation services including Google Translate, Apertium and several others.
  • Users can edit translations to make them better. You decide who can translate.
  • Professional translations are available from SpeakLike and other professional translation agencies for a few cents per word.
  • Easy to install and manage
